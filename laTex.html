Ознаки Даламбера

Нехай \(\sum\limits_{n = 1}^\infty {{a_n}}\) − ряд з додатніми членами. Тоді справедливим є наступне:

  • Якщо \(\lim\limits_{n \to \infty } {\large\frac{{{a_{n + 1}}}}{{{a_n}}}\normalsize} < 1,\) то ряд \(\sum\limits_{n = 1}^\infty {{a_n}}\) збіжний ;

  • Якщо \(\lim\limits_{n \to \infty } {\large\frac{{{a_{n + 1}}}}{{{a_n}}}\normalsize} > 1,\) то ряд \(\sum\limits_{n = 1}^\infty {{a_n}}\) розбіжний;

  • Якщо \(\lim\limits_{n \to \infty } {\large\frac{{{a_{n + 1}}}}{{{a_n}}}\normalsize} = 1,\) то ряд \(\sum\limits_{n = 1}^\infty {{a_n}}\) В цьому випадку необхідно розв'язувати іншим методом.

    • 1.Дослідити ряд на збіжність \(\sum\limits_{n = 1}^\infty {\large\frac{{{3^n}}}{{{n^2}}}\normalsize}.\)

    Розв'язання:

    За ознакою Даламбера:

    \[ {\lim\limits_{n \to \infty } \frac{{{a_{n + 1}}}}{{{a_n}}} } = {\lim\limits_{n \to \infty } \frac{{\frac{{{3^{n + 1}}}}{{{{\left( {n + 1} \right)}^2}}}}}{{\frac{{{3^n}}}{{{n^2}}}}} } = {\lim\limits_{n \to \infty } \left[ {\frac{{{3^{n + 1}}}}{{{3^n}}} \cdot \frac{{{n^2}}}{{{{\left( {n + 1} \right)}^2}}}} \right] } = {\lim\limits_{n \to \infty } \left[ {3{{\left( {\frac{n}{{n + 1}}} \right)}^2}} \right] } = {3\lim\limits_{n \to \infty } {\left( {\frac{{n + 1 - 1}}{{n + 1}}} \right)^2} } = {3\lim\limits_{n \to \infty } {\left( {1 - \frac{1}{{n + 1}}} \right)^2} = 3.} \]
